CMIS: Building web applications with Javascript

This web application contains the code samples for appendix D of the book 'CMIS in Action'. It is divided into several sections each on illustrating a certain aspect of CMIS. To run the sample code deploy this application in a web server (e.g. Tomcat). YOu should have a CMIS server running and imported the sample data from the sample application The Blend.
